## 🚀 Featured projects

### 🛰️ [MCP-Packet-Tracer](https://github.com/Mats2208/MCP-Packet-Tracer) &nbsp;·&nbsp; ⭐ 95 &nbsp;·&nbsp; [live demo](https://www.mcpnetwork.top/)
AI-powered Cisco Packet Tracer automation over **MCP**. Describe a network in natural language and it
generates, validates and deploys full topologies — OSPF, EIGRP, RIP, DHCP. &nbsp;`Python`

### 🧠 [hyprdesk](https://github.com/Mats2208/hyprdesk)
Orchestrate a team of AI coding agents on your desktop. A router-leader delegates to workers over a
local **MCP tunnel (A2A)**, with git-worktree isolation and merge-back, wrapped in a **Tauri v2** IDE surface. &nbsp;`TypeScript`

### 🤖 [ao](https://github.com/Mats2208/ao)
Agnostic agent orchestrator: a brain/router (`claude` / `codex` / `opencode`) delegates real coding
tasks to worker CLI agents in **isolated git worktrees** — real, separate processes, not internal subagents. &nbsp;`JavaScript · Tauri`

### 🎬 [FestCine](https://github.com/Mats2208/FESTCINE) &nbsp;·&nbsp; [live demo](https://festcine.matsoto.dev)
Full-stack festival management where **all business logic lives in the database** (SQL Server stored
procedures, triggers, transactions). FastAPI is a thin translation layer; React on top. &nbsp;`FastAPI · React · SQL Server`

### 📈 [EDOLab](https://github.com/Mats2208/EDO-FRONTEND)
Interactive platform to visualize and solve **Ordinary Differential Equations** with Euler and
Runge-Kutta (RK4), AI-assisted. &nbsp;`React · FastAPI`

### 🎥 [YouTube Downloader](https://github.com/Mats2208/DownloadYoutubeTool)
Native **macOS** app to download YouTube videos up to 4K with automatic stream merge. &nbsp;`Swift · SwiftUI`
