## 🇪🇸 Español

### Sobre mí

Soy Mateo, desarrollador de software y estudiante de ingeniería en la UPSA. Paso la mayor parte del tiempo construyendo herramientas alrededor de la IA: cosas como servidores **MCP** para que los modelos puedan operar software real, y orquestadores que ponen a varios agentes de código a trabajar en paralelo.

Me gusta empezar de una idea simple y llevarla hasta algo que de verdad se pueda usar, no solo un demo bonito. Trabajo cómodo tanto en el backend como armando interfaces, y últimamente me interesa bastante todo lo que tenga que ver con automatización y agentes.

**Lenguajes:** Python · TypeScript · JavaScript · C++ · C# · Swift · Dart
**Backend & apps:** Node.js · Flask · Flutter · Tauri · Firebase
**Herramientas:** Git · GitHub Actions · MCP

## 🇺🇸 English

### About me

I'm Mateo, a software developer and engineering student at UPSA. Most of my time goes into building tooling around AI: things like **MCP** servers that let models actually operate real software, and orchestrators that put several coding agents to work in parallel.

I like taking a simple idea and pushing it until it's genuinely usable, not just a nice demo. I'm comfortable on the backend and building interfaces alike, and lately I'm really into anything involving automation and agents.

### What I work with

**Languages:** Python · TypeScript · JavaScript · C++ · C# · Swift · Dart
**Backend & apps:** Node.js · Flask · Flutter · Tauri · Firebase
**Tools:** Git · GitHub Actions · MCP

### Projects

**[MCP-Packet-Tracer](https://github.com/Mats2208/MCP-Packet-Tracer)** &nbsp;·&nbsp; ![stars](https://img.shields.io/github/stars/Mats2208/MCP-Packet-Tracer?style=flat-square&label=%E2%98%85)
An MCP server that lets you drive **Cisco Packet Tracer** with plain language. You describe the network you want and it builds the full topology —OSPF, DHCP, EIGRP, RIP— validates it and deploys it. It's my longest-running project and the one I've learned the most from.

**[hyprdesk](https://github.com/Mats2208/hyprdesk)**
A desktop app (Tauri v2, macOS) for coordinating several AI coding agents at once. A lead agent hands work out to others that each run in their own `git worktree`, then merges the changes back — all from a single terminal/IDE-style surface.
